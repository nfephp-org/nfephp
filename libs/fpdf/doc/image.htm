Image

Image(string file [, float x [, float y [, float w [, float h [, string type [, mixed link]]]]]])

Descrição

Coloca uma imagem na página. As dimensões podem ser especificadas de diferentes formas:
  • largura e altura explicitas (informadas na unidade selecionada pelo usuário)
  • uma dimensão explicita, a outra será calculada automaticamente de forma a manter a proporção original
  • sem dimensões explicitas, neste caso a imagem será colocada em 72 dpi
Os formatos suportados são JPEG, PNG e GIF. A extensão GD é requerida para GIF.

Para JPEG, todos os tipos são permitidos:
  • gray scales (tons de cinza)
  • true colors (24 bits)
  • CMYK (32 bits)
Para PNG, são permitidos:
  • tons de cinza com pelo menos 8 bits (256 níveis)
  • cores indexadas
  • true colors (24 bits)
mas não são suportados:
  • Interlacing
  • Alpha channel
Para GIF: no caso de um GIF animado, só o primeiro frame vai ser usado.

Se uma cor transparente é definida, ela vai ser levada em conta.

O formato pode ser especificado ou inferido pela extensão do arquivo.
É possível colocar um link na imagem.

Nota: se a mesma imagem é usada várias vezes, só uma cópia vai ser embutida no arquivo.

Parâmetros

file
Nome completo ou URL do arquivo que contém a imagem.
x
Abscissa do canto superior-esquerdo. Se não for informada ou igual a null, a abscissa atual será usada.
y
Ordenada do canto superior-esquerdo. Se não for informada ou iguak a null, a ordenada atual será usada; em alguns casos uma quebra de página será disparada antes (no caso de quebra de página automática habilitada) e, depois da chamada, a ordenada corrente é movida para a parte inferior da imagem.
w
Largura da imagem na página. Se não informada ou zero, ela será calculada automaticamente.
h
Altura da imagem na página. Se não informada ou zero, ele será calculada automaticamente.
type
Formato da imagem. Os possíveis valores são (diferenciando maiúsculas e minúsculas): JPG, JPEG, PNG e GIF. Se não informado, o tipo será inferido pela extensão do arquivo.
link
URL ou identificador retornado pela função AddLink().
