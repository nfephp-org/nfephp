Link

Link(float x, float y, float w, float h, mixed link)

Descrição

Põe um link sobre uma área retangular da página. Os links do texto ou da imagem são postos geralmente através de Cell(), Write() ou Image(), mas este método pode ser útil, por exemplo, para definir uma área clicável dentro de uma imagem.

Parâmetros

x
Abscissa do canto superior-esquerdo do retângulo.
y
Ordenada do canto superior-esquerdo do retângulo.
w
Largura do retângulo.
h
Altura do retângulo.
link
URL ou identificador retornado por AddLink().
