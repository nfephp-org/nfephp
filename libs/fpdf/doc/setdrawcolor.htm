SetDrawColor

SetDrawColor(int r [, int g, int b])

Descrição

Define uma cor para ser usada em todas as operações de desenho (linhas, retângulos e bordas de células). Ela pode ser informada como componentes RGB ou tons de cinza. O método pode ser chamado antes antes que a primeira página seja criada e o valor será mantido de uma página para outra.

Parâmetros

r
Se g e b estiverem preenchidos, informa o componente vermelho; caso contrário, indica o tom de cinza. Valores entre 0 e 255.
g
Componente verde (entre 0 e 255).
b
Componente Azul (entre 0 e 255).
