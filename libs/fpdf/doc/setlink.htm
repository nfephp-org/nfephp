SetLink

SetLink(int link [, float y [, int page]])

Descrição

Define a página e a posição para onde um link será apontado.

Parâmetros

link
Um identificador de link retornado por AddLink().
y
Ordenada da posição do alvo; -1 indica a posição atual. O valor padrão é 0 (início da página).
page
Número da página do alvo; -1 indica a página corrente. Este é o valor padrão.
